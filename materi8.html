Ringkasan Materi

Dari fondasi machine learning hingga neural network modern — pelajari bagaimana sistem buatan dapat belajar, beradaptasi, dan membuat keputusan cerdas. Materi ini terdiri dari 3 topik utama yang saling berkaitan dan dirancang untuk memberikan pemahaman menyeluruh.

Pengantar Kecerdasan Buatan

Kecerdasan Buatan (AI) adalah simulasi kecerdasan manusia pada mesin yang diprogram untuk berpikir dan bertindak layaknya manusia.

  • Sejarah dan perkembangan AI
  • Narrow AI vs General AI vs Super AI
  • Turing Test dan konsep kecerdasan
  • Cabang AI: ML, NLP, Computer Vision

Machine Learning

Machine Learning memungkinkan komputer belajar dari data tanpa diprogram secara eksplisit.

  • Supervised vs Unsupervised Learning
  • Regresi Linier dan Logistik
  • Decision Tree dan Random Forest
  • K-Means Clustering dan K-NN

Deep Learning & Neural Network

Deep Learning menggunakan jaringan saraf tiruan berlapis untuk memproses data dengan cara yang meniru otak manusia.

  • Arsitektur Neural Network
  • Backpropagation dan Gradient Descent
  • CNN untuk Computer Vision
  • RNN dan Transformer (BERT, GPT)
